# 地球物理期末作品集 · Interactive Portfolio

🎉 歡迎來到地球偵探社！我們專門解鎖地球的秘密，今天，讓我們用五大關卡一起挑戰地球物理的宇宙，並且（當然）做一些互動的超級酷炫小遊戲！🌍

## 🕵️‍♀️ 你將會學到什麼？

在這個超酷的地球物理網頁中，我們會帶你進入五個神秘的領域，每個領域都能讓你感覺自己像是一位地球偵探：

1️⃣ **地球作為一顆行星** 🪐  
2️⃣ **重力與大地測量** ⚖️  
3️⃣ **地震學與地球內部** 🌋  
4️⃣ **年齡、熱與電性** 🔥  
5️⃣ **地磁與古地磁** 🧭

每一個領域都有簡單的介紹、互動小遊戲和小測驗，來幫助你檢驗自己是否準備好成為地球的超級偵探！💡

## 📚 每一關都讓你玩得開心、學得輕鬆

- **Geophysics Playground**: 第一個挑戰是用五張卡片讓你理解地球的基本概念，還有小測驗來檢查你學了多少。簡單又有趣，還能挑戰你能不能順利解鎖下一關！🎮

- **重力異常計算器**: 這裡可以讓你像專業的重力勘探員一樣，計算自由空氣異常和布格異常，了解地底密度如何影響重力。當然，還可以帶著工具，一起玩玩看。🎲

- **校園震測實驗**: 想當地震學家嗎？這部分會教你如何在校園內設置震源、讀取走時圖，並估算地層厚度，還有互動工具可以幫助你估算！快來動手吧！🔨

- **PyGMT & Scotia Plate**: 使用 PyGMT 畫出 Scotia Plate 的板塊邊界與地形，然後展示各種地震與火山活動，讓你了解地球板塊如何在這片區域上演戲！🌋🗺️

## 🤔 怎麼使用？

1. **開啟網頁**  
   只要打開 `index.html`，你就能在瀏覽器中體驗這些地球物理的知識和互動遊戲了。沒有安裝任何東西，也不需要任何準備！⏳

2. **檔案結構**  
   - `index.html`: 這是整個網頁的主檔，包含所有互動內容和介紹。
   - `scotia_plate_map.png`: 你會看到的板塊邊界地圖。
   - `taiwan_map.png`: 台灣及周邊海底地形圖。
   - 其他圖片：有校園震測的裝備大合照、測站佈設過程圖等，讓你感覺自己就像是地球科學的現場觀察員。📸

3. **互動功能**  
   點擊各個主題，你可以選擇不同的選項來進行計算或回答小測驗，系統會即時給出反饋。還有許多圖片可以點擊查看詳情，讓你更深入了解地球的秘密！💥

## 💻 技術架構

這個網頁專案有點像是地球物理與互動遊戲的混合體，使用了以下技術來讓這一切發生：
- **HTML**：結構部分，沒它這個網頁就不存在啦。
- **CSS**：讓這個網頁看起來不那麼呆板，這就像是給你的地球物理課本加上了一層酷炫的外衣。😎
- **JavaScript**：讓一切互動生動起來，比如重力異常計算器和小測驗等，沒有它，這裡就只是個靜態頁面而已。💥
- **PyGMT**：這個工具幫助我們繪製漂亮的板塊邊界圖，還能呈現全球地形資料，讓我們探索 Scotia Plate 時不會迷路！🌍

## 🚀 快速開始

只要打開 `index.html` 文件，你就能開始這段學習旅程，點擊不同的分頁探索地球物理的奧秘。你甚至能用這些互動小工具來挑戰自己，看你能不能解答每一題！每個工具都有詳細的指引，還能幫助你在玩中學、學中玩。
